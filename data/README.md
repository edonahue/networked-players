# Data workspace

This directory contains only public contracts, schemas, documentation, and synthetic fixtures.

Private collection exports, raw account-linked responses, generated full catalogs, database files, and production snapshots are ignored and must remain outside Git.

Planned layers:

1. source records with provenance;
2. normalized releases, contributors, roles, and credits;
3. versioned analytical tables;
4. graph snapshot manifests;
5. static challenge artifacts.

Every published artifact should identify its source version, schema version, generation method, and rights posture.
